Settings on [[roam/excalidraw]]

Fullscreen margin

  • :full-screen-margin
    • 0 will make the drawing cover the entire Roam window
    • Default setting is 0.015 leaving a thin space around the edited drawing so you "don't forget" that you are in Roam.

Embedded image

  • :max-embed-width
    • is the maximum width of an image, the image will be scaled to the available space but maximum this size.
    • Default is 600
  • :max-embed-height
    • Default is 400
  • :img
    • Valid values are "SVG" and "PNG"

Excalidraw Theme

  • :mode
    • Default drawing mode: "dark" or "light"

Outlining support

  • :nested-text-start-left
    • The left position of the first column of text. Default is 320 to avoid the drawing pannel from appearing on top of the text.
  • :nested-text-start-top
    • Default value is 40
  • :nested-text-rows
    • the number of nested blocks that will appear one under the other on the drawing. When the limit is reached a new column is started
  • :nested-text-row-height
    • Vertical distance between rows. Default value is 40.
  • :nested-text-col-width
    • Default value is 400. Number of pixels between columns.
  • :nested-text-font-size
    • Size of the font used on the drawing. Default is 20.
  • :nested-text-font-family
    • Valid values are 1,2 and 3. The 3 fonts available in Excalidraw.
